中文支持论坛

您可以使用“主题编辑器”来编辑主题中包含的 CSS 和 PHP 文件。

要开始，请在右上角选择一个主题，并点击“选择”。然后在出现的列表中点击要编辑的文件，文件就会被加载在编辑区域中。

对于 PHP 文件，您可以使用“文档”下拉列表选择我们从文件中识别出的函数。点击“查询”按钮可查看有关该函数的页面。

当编辑完成时，请点击“上传文件”。

建议：若您编辑您正在使用的主题，编辑不当则可能造成站点的故障。

升级新版本的主题将会导致您所做的修改被覆盖。要避免类似悲剧的发生，请考虑自建一个子主题。
